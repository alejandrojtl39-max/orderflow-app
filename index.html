🤖 Groq API Key GRATIS

Necesaria para extraer productos del PDF con IA. 100% gratis, sin tarjeta de crédito.

📋 Cómo obtener tu API key gratis (60 segundos):
1. Ve a console.groq.com
2. Clic en "Sign Up" → puedes usar tu cuenta de Google
3. En el panel ve a "API Keys"
4. Clic en "Create API Key"
5. Copia la key (empieza con gsk_...)
6. Pégala aquí y guarda ✅
